# Portfolio v2

Personal portfolio showcasing software engineering projects focused on mobile development, backend systems, and applied artificial intelligence.

Built with React, TypeScript, and Vite, the portfolio serves as the central hub for presenting academic, personal, and research projects, including mobile applications, backend APIs, and AI-related work.

🌐 Live Website

[cristianburgoin.dev](https://cristianburgoin.dev)

---

## Features

- English and Spanish localization
- Custom type-safe translation system
- Dark and light themes
- Persistent user preferences
- Interactive project showcase
- Mobile application previews
- APK download integration
- GitHub repository integration
- Responsive design
- Downloadable resumes
- Modern single-page portfolio architecture

## Running Locally

### Requirements

- Node.js 20+
- npm

### Setup

```bash
git clone https://github.com/cburgoin-dev/portfolio-v2.git

cd portfolio-v2

npm install

npm run dev
```
